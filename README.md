# Bible Memorizer

A simple web-based Bible memorization application built with React and Tailwind CSS.

This app allows users to load a Bible passage and practice memorization using three different modes:

- Hide: Gradually hides words from the verse.
- Pick: Select the correct next word from multiple choices.
- Type: Type the first letter of each word in order.

## Features

- Supports different Bible versions
- Dynamic book and chapter loading
- Verse range selection (example: 1-3)
- Multiple memorization modes
- Progress bar tracking
- Word mistake heatmap
- Mobile-friendly responsive layout

## Technologies

- React (via CDN)
- Tailwind CSS
- Font Awesome
- Vanilla JavaScript

## Deployment

This project is deployed using GitHub Pages:

https://alexanderortizcuellar.github.io/bible-memorizer

## Data Source

Verse data is loaded dynamically from a separate GitHub repository endpoint.

## Usage

1. Select Bible version.
2. Choose book and chapter.
3. Enter verse number or range.
4. Click "Start Learning".
5. Choose a memorization mode.
